#  AI-Powered Resume Analyzer & Job Matcher

##  Overview
A full-stack web application that analyzes resumes and suggests job roles based on extracted skills. It also provides feedback on missing skills and tracks user history.

---

##  Features
-  Upload Resume (PDF)
-  Skill Extraction (Mock NLP logic)
-  Job Role Suggestions
-  Missing Skills Detection
-  User Authentication (JWT)
-  Resume History Tracking
-  Logout functionality

---

##  Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Vanilla)

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Tokens)
- bcrypt
